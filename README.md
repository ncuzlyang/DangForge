# DangForge｜MR Dang 价值投资技能

把 MR Dang 的《价值投资功法》，整理成 AI Agent 可以执行的中文投资分析流程。

它不是荐股工具，也不预测涨跌。它做的事情很简单：让 AI 在分析股票前，先把该核验的核验、该计算的计算、该提醒的提醒。

```text
拉最新数据
→ 查公告 / 财报 / 研报交叉核验
→ 看商业模式：谁求谁
→ 拆利润变量
→ 算静态估值和前瞻市盈率
→ 做悲观 / 中性 / 乐观三档
→ 看预期股息率和现金流
→ 给仓位计划和风险触发条件
```

故事是讲给别人听的，钱可是自己的。DangForge 的目标，就是让 AI 少写一点顺滑小作文，多跑一遍功法。

> **免责声明**：本项目仅用于教育、研究和投资分析方法学习，不构成任何投资建议、证券推荐或收益承诺。使用者应自行核验数据、独立判断风险，并对自己的投资决策负责。

## 它会重点检查什么

- **数据核验**：股价、市值、市盈率、市净率、股息率、财报、公告、研报预测，不能凭印象写。
- **商业模式**：先看谁求谁，再看财报和估值。
- **前瞻估值**：不能只看静态市盈率，必须算本年度前瞻市盈率，并做悲观 / 中性 / 乐观三档。
- **预期股息率**：看未来可能分多少，不只看过去分过多少。
- **仓位纪律**：买多少、怎么加、什么情况降级，都要提前写清楚。
- **风险证伪**：什么事实出现，说明原逻辑错了。
