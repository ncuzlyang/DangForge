# 商业模式：先看谁求谁

## 一句话

商业模式，一言以蔽之，就是一个字：**求**。

投资时先别急着打开财报。先看公司和上下游、消费者、政府、资本之间，到底是谁有求于谁。

求的外在表现，就是话语权和定价权。

## 四档商业模式

### 三求

上游求、下游求、政府求。

这类企业是响当当的好商业模式。上游挤破脑袋想合作，下游排队买产品，当地政府也希望你别搬走。

估值可以给溢价，但仍要核验利润、现金流和价格。

### 两求

至少两个关键相关方有求于公司。

典型如部分资源企业：上游资源/矿工弱，政府重视税收和就业，下游随行就市。这类企业往往不是永远舒服，但在周期正确时弹性很强。

### 一求

大多数普通公司都在这一档。和上游平等谈，和下游公平竞争，政府看在税收份上呵护。

能活，但不一定强。估值不能乱给。

### 零求

上游不惯着你，下游不买账，政府也不求你。

这类公司基本面再会讲故事，也要小心。低 市盈率 可能不是便宜，是陷阱。

## 分析路径

```text
公司卖什么
→ 上游是谁
→ 下游是谁
→ 政府/监管如何影响它
→ 谁求谁
→ 定价权在哪边
→ 利润分配权在哪边
→ 财报是否验证这个判断
```

## 输出要求

分析商业模式时，不要只说“护城河深”“壁垒高”。要说清楚：

- 谁离不开公司？
- 公司离不开谁？
- 涨价时谁先受益？
- 成本上升时谁先受伤？
- 行业下行时谁还能活得舒服？

同样商业模式下，讨论估值才有意义。不同商业模式不能只拿 市盈率/市净率 横向硬比。
