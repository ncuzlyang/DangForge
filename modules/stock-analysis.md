# 个股分析流程

## 一句话

分析个股，不是问“能不能买”，而是先问“它凭什么赚钱”。

## 标准流程

```text
一句判断
→ 数据核验
→ 天阶/地阶/玄阶归类
→ 商业模式：谁求谁
→ 利润来源
→ 关键变量
→ 财报验证
→ 估值边界
→ 前瞻 市盈率 三情景
→ 分红/现金流
→ 仓位计划
→ 风险证伪
→ 最终判断
```

## 关键变量提取

不要淹死在会计科目里。每家公司只抓 2–4 个真正影响利润的变量。

- 资源公司：价格、产量、成本、资本开支。
- 银行：净息差、不良率、拨备、规模、分红、资本充足率。
- 消费：人口、渗透率、客单价、毛利率、渠道费用。
- 制造：产能、单价、成本、良率、订单。
- 平台：用户、频次、货币化率、补贴、监管。
- 成长股：收入增速、利润兑现、费用率、估值消化。

## 财报验证

财报是皮，经营是骨。

财报要用来验证商业逻辑，不是让你在里面找一个漂亮数字然后自我感动。

检查：

- 收入结构是否支持主逻辑？
- 毛利率变化是否能解释？
- 经营现金流是否跟得上利润？
- 负债是否可控？
- 分红是否可持续？
- 股权激励是否给出利润指引？
- 资本开支是否会改变未来现金流？

## 降级条件

- 收入结构不支持主逻辑。
- 利润和现金流长期背离。
- 控股股东风险明显。
- 行业竞争激烈且公司没有成本/资源/渠道优势。
- 分红不可持续但估值依赖股息率。
- 当前市值已经把乐观情景打满。
