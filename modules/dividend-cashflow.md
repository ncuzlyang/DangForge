# 股息、分红和现金流

## 一句话

股息率可以是安全绳，但高股息本身不是安全。

先纠正一个观点：要算的是**预期股息率**，不是过去股息率。

过去分出去的钱，和现在买入的人没有关系。拿去年每股分红直接除以当前股价，只能当粗略参考，不能当投资结论。

## 核心公式

说人话：

```text
公司赚 100 块，分给股东 30 块，就是 30% 派息率。
你花 100 块买股票，一年拿 5 块分红，就是 5% 股息率。
```

正式公式：

```text
预期每股分红 = 预期每股收益 每股收益 × 预期派息率
预期股息率 = 预期每股分红 ÷ 当前股价
```

或者：

```text
预期股息率 = 预期归母净利润 × 预期派息率 ÷ 当前总市值
```

## 五步股息率法

### 第一步：分类讨论

先把公司分成四类。分类不是绝对的，是实时变化的。

- A 类：盈利稳定、派息稳定。代表：银行、公用事业、电信、铁路、成熟消费龙头等。
- B 类：派息率稳定，但利润周期波动。代表：海运、煤、有色、化工、部分周期消费等。
- C 类：利润相对稳定，但派息不稳定。代表：需要大额资本开支、并购、扩产的公司。
- D 类：高股息但业务衰退，或者分红不可持续。新手尽量不要碰，怕你把握不住。

A 类确定性强，通常股息率低一些。B/C 类确定性差一些，需要更高股息率补偿。若 A 类企业股息率也高，就值得重点研究。

### 第二步：测算基准

先用已核验数据测一个基准。

```text
最近两年平均派息率
× 本年度预期 每股收益
÷ 当前股价
= 预期基准股息率
```

本年度 每股收益 不能拍脑袋。优先用程序拉取最新财务数据、公告、研报预测；没有研报时，用前三季度/半年度利润年化，并调整季节性和一次性损益。

### 第三步：行业/周期基准纠正

算完基准，还只是会计。要成为价值投资者，还要做行业纠正。

- A 类：盈利和分红惯性强，一般不需要大幅纠正，除非有考核目标、管理层表态、资本监管变化。
- B 类：周期强。周期顶部可能下调派息率，因为公司会屯现金；周期底部反而可能上调预期，用分红平滑周期。
- C 类：看资本开支。若公告里有大额投资、并购、扩产，要下调派息率；若出售资产、天降横财，可适度上调。
- D 类：不要因为表面股息率高就冲进去。先问业务是不是在衰退。

### 第四步：大股东诉求纠正

大股东如果急需现金，又没有减持，很可能通过提高派息率拿钱。

重要财务投资者有协议或回报诉求时，也可能提高分红预期。

但这一步必须有证据：股东背景、公告、历史分红、资金需求、公开表态。不能纯脑补。

### 第五步：公告纠正

公告确定性最高，放在最后。

如果公司明确承诺：

```text
派息率不低于某比例
或现金分红不低于某金额
或保持分红连续性和稳定性
或提高分红水平
```

就要用公告对前面测算结果进行最后校准。

## 分红检查

```text
历史分红是否稳定？
派息率是否合理？
利润能不能覆盖分红？
经营现金流能不能覆盖分红？
资本充足率/负债压力是否限制分红？
控股股东是否有现金需求？
公告有没有明确分红承诺？
行业下行时分红还能不能维持？
```

## 股息率用法

- 4%：高股息观察线。
- 5%：组合防守收益率目标。
- 6% 以上：有吸引力，但必须检查是否股息陷阱。

股价跌、分红不变，股息率上升，复投能降低成本。股价涨太多，股息率下降，可能要换到更高赔率资产。

## 银行股特别检查

银行股便宜不能只看 市净率。

必须看：

```text
不良率
拨备覆盖率
净息差
ROE
核心一级资本充足率
分红率
资产扩张质量
```

低 市净率 如果配低 ROE，可能只是合理低估。低 市净率 + 资产质量改善 + 分红稳定，才有修复意义。
