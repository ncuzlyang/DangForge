# 邪修功法：投机和事件

## 一句话

多看，多了解；少动，少参与。

邪修不是主修。它可以研究，但必须小仓，必须知道自己在玩什么游戏。

## 可研究的事件

- 突发地缘冲突。
- 自然灾害影响供给。
- 超预期政策变化。
- 产业链突发断供。
- 重组、并购、回购、分红变化。
- 市场错误定价的短期套利。

## 不能碰或少碰

- 全网都知道的热点。
- 已经涨了一大段的题材。
- 流动性差的小票。
- 基本面差但只靠故事硬拉。
- 需要接最后一棒的博弈。

## 邪修三问

```text
这个事件是不是突发？
我和大资金是否还在同一起跑线？
错了之后能不能体面退出？
```

## 仓位要求

- 默认 1%–3%。
- 不得挪用核心仓位。
- 不得因为短线刺激破坏长期组合。
- 事件兑现前后必须复盘退出条件。
