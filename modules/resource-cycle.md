# 资源周期：价格只是结果，储量才是底牌

## 一句话

金融是表象，资源是底层。

资源股不要先看股价，先看资源禀赋。你不能卖出一个你没有的东西。

## 分析顺序

```text
有没有资源
→ 资源质量如何
→ 产能和产量多少
→ 成本曲线位置
→ 价格周期位置
→ 供给有没有天花板
→ 需求有没有刚性或增量
→ 利润弹性怎么算
→ 分红和现金流能不能跟上
```

## 关键变量

- 资源价格：铜、铝、金、煤、油、化肥等。
- 产量：能不能稳定生产，能不能扩产。
- 成本：电力、矿石、运输、人工、折旧。
- 资本开支：赚的钱是不是都拿去扩产。
- 政策约束：产能天花板、环保、能耗、矿权。
- 库存和供需：补库还是去库，短缺还是过剩。

## 有色特别注意

很多“有色公司”其实没有矿，只是加工商。那不叫资源底牌，那叫代工。

有矿、有电、有成本优势、有产能天花板保护，才有资格谈资源重估。

## 估值方法

资源股不能只看当期 市盈率。

必须做三情景：

```text
悲观：资源价格回落，利润收缩
中性：当前价格中枢维持
乐观：价格上涨或成本下降，利润弹性释放
```
