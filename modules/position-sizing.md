# 仓位控制

## 一句话

仓位控制，是把资金和心理绑在一起的艺术。

很多人亏钱，不是因为完全看错，而是因为仓位错了。看对小仓，没意义；看错重仓，要命。

## 核心原则

- 仓位控制是必修课，先活下来，再谈复利。
- 不要为了回本而补仓。
- 不要卖掉强资产去救弱资产。
- 加仓必须拉开价格距离，不能被浮亏绑架。
- 止盈不是背叛长期主义，而是把资金挪到更高赔率的位置。
- 分红复投是普通投资者少数能长期执行的安全策略。

## 仓位标签

- 0%：避开。
- 1%–3%：观察仓/期权仓。
- 5%：小而有意义。
- 10%：中等配置，需要逻辑完整。
- 15%：强配置，需要估值和确定性同时舒服。
- 20%：单一非银行个股上限，必须极高确定性。

## 买前八问

```text
我买的是什么逻辑？
关键变量是什么？
买入区在哪里？
加仓区在哪里？
减仓区在哪里？
错了什么信号出现要走？
最大能亏多少？
这个仓位会不会影响睡觉？
```

## 持有纪律

- 日内波动不能推翻计划。
- 基本面没坏，价格下跌才有加仓意义。
- 基本面坏了，价格下跌不是机会，是提醒。
- 每次加仓要有距离，比如每下跌 5% 或 10% 才重新判断，而不是天天补。
